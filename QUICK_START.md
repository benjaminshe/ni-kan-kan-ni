# 🚀 快速开始：在华为手机上安装"你看看你"

## 一键构建（推荐新手）

如果你不熟悉 Android 开发，这是最简单的方法：

### 第 1 步：注册 Expo 账户
1. 打开 https://expo.dev
2. 点击"Sign up"注册
3. 使用邮箱或 GitHub 账户登录

### 第 2 步：安装构建工具
打开电脑终端（命令行），执行：

```bash
npm install -g eas-cli
```

### 第 3 步：登录 Expo
```bash
eas login
```
输入你刚才注册的 Expo 账户信息

### 第 4 步：进入项目目录
```bash
cd /workspace/projects/client
```

### 第 5 步：构建 APK
```bash
eas build --platform android --profile preview
```

### 第 6 步：等待构建完成
- 构建时间通常 10-20 分钟
- 构建完成后会显示下载链接

### 第 7 步：下载 APK
1. 在浏览器中打开构建日志中的下载链接
2. 下载 `nikanikani.apk` 文件

---

## 在华为手机上安装

### 方法 1：USB 传输（推荐）

1. **用 USB 线连接手机和电脑**
2. **在手机上选择"文件传输"模式**
3. **将 APK 文件复制到手机的"下载"文件夹**
4. **在手机上打开"文件管理器"**
5. **找到并点击 APK 文件**
6. **点击"安装"**
7. **如果提示"允许安装未知应用"，点击"前往设置"并允许**
8. **完成安装**

### 方法 2：通过云盘传输

1. **将 APK 上传到云盘（百度网盘、阿里云盘等）**
2. **在手机云盘 App 中下载 APK**
3. **点击安装**

## 安装后的设置

### 1. 授予网络权限（首次运行时）
- 打开"你看看你"应用
- 允许网络访问权限
- 这样应用才能连接后端服务器

### 2. 关闭纯净模式（如果提示）
- 设置 -> 系统和更新 -> 纯净模式
- 选择"退出"

## 常见问题

### ❓ 为什么安装失败？

**可能原因：**
1. 手机存储空间不足 → 删除一些文件
2. 旧版本未卸载 → 先卸载旧版本
3. 文件损坏 → 重新下载 APK

### ❓ 应用无法连接服务器？

**解决方法：**
1. 检查手机是否联网
2. 打开 Wi-Fi 或移动数据
3. 关闭防火墙或 VPN

### ❓ EAS Build 失败怎么办？

**解决方法：**
1. 检查 Expo 账户是否正常
2. 确保网络连接稳定
3. 查看构建日志中的错误信息
