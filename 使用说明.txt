这个文件夹里的东西是什么？
- 这是一个手机 APP 的源代码
- 叫做"你看看你"，用来统计手机使用情况
- 你需要把它安装到你的华为手机上

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第一步：准备工具】（需要 30-40 分钟）

你需要安装 3 个软件，下载地址如下：

1. Node.js
   下载地址：https://nodejs.org/
   安装方法：双击下载的文件，一路点击"下一步"即可

2. Java (JDK 17)
   下载地址：https://adoptium.net/temurin/releases/?version=17
   安装方法：双击下载的文件，一路点击"下一步"即可

3. Android Studio
   下载地址：https://developer.android.com/studio
   安装方法：
   - 双击下载的文件，一路点击"下一步"
   - 安装完成后，打开 Android Studio
   - 选择"Standard"安装类型
   - 等待下载完成（这需要 15-30 分钟，需要联网）

⚠️  注意：
- 安装 Android Studio 时会下载很多文件，需要稳定的网络
- 这个步骤只需要做一次
- 如果已经安装过这些软件，可以跳过

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第二步：下载并解压】

1. 下载 nikanikani-project.zip 文件（约 3.5MB）
2. 解压到任意位置，比如 D:\nikanikani\
3. 解压后，你会看到以下文件：
   - 一键构建.bat          ← 这个是重点！
   - client 文件夹
   - server 文件夹
   - package.json
   - 使用说明.txt          ← 你正在看的文件

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第三步：双击运行】

1. 打开解压后的文件夹（比如 D:\nikanikani\）
2. 找到"一键构建.bat"这个文件
3. 双击它
4. 黑色窗口会自动运行，耐心等待（20-40 分钟）
5. 看到提示"构建完成"后，按任意键
6. 会自动打开包含 APK 文件的文件夹

⚠️  注意：
- 构建过程需要联网
- 首次构建需要下载很多文件，会比较慢
- 如果中途失败，可以重新双击"一键构建.bat"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【第四步：安装到手机】

构建成功后，你会得到一个 APK 文件：
- 文件名：app-debug.apk
- 位置：会自动打开的文件夹里

安装方法：

方法一：通过 USB 连接（需要手机开启 USB 调试）
1. 用 USB 线连接手机和电脑
2. 在手机上打开"设置" → "关于手机"
3. 连续点击"版本号"7次，开启开发者模式
4. 打开"设置" → "系统和更新" → "开发者选项"
5. 打开"USB 调试"开关
6. 在电脑上打开命令提示符（搜索"cmd"）
7. 输入命令：adb install <APK文件路径>
8. 如果提示"Success"，说明安装成功

方法二：手动安装（更简单）
1. 将 app-debug.apk 文件发送到手机（微信、QQ、U盘都可以）
2. 在手机上找到这个文件
3. 点击安装
4. 如果提示"禁止安装未知来源应用"
5. 按提示去设置里允许，然后再安装

【遇到问题怎么办？】

问题 1：双击"一键构建.bat"后，提示"未检测到 Node.js"
解决：说明还没安装 Node.js，参考【第一步】安装

问题 2：双击"一键构建.bat"后，提示"未检测到 Java"
解决：说明还没安装 Java，参考【第一步】安装

问题 3：双击"一键构建.bat"后，提示"未检测到 adb"
解决：说明还没安装 Android Studio，参考【第一步】安装
      不过这个只是警告，不影响构建，可以继续

问题 4：构建过程中卡住不动
解决：检查网络连接是否正常，可能正在下载文件
      耐心等待，最长可能需要 40 分钟

问题 5：构建失败，提示错误
解决：
      1. 确保网络连接正常
      2. 确保磁盘空间足够（至少 10GB）
      3. 尝试重新双击"一键构建.bat"

问题 6：APK 安装失败
解决：
      1. 检查手机存储空间
      2. 确保手机允许安装未知来源应用
      3. 尝试手动安装方法

- 双击构建：20-40 分钟
- 安装到手机：5 分钟
- 总计：58-118 分钟（约 1-2 小时）

- 如果已经安装过开发工具，只需要 30-50 分钟
- 第二次构建会快很多（5-10 分钟）
